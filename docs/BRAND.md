# BRAND

## Identita

Atletico Xeneizes deve comunicare:

- premium
- elegante
- sportivo
- moderno
- minimalista
- ordinato
- essenziale
- ispirazione scandinava nelle proporzioni e negli spazi

## Palette

Colori ufficiali di riferimento:

- Nero principale: `#0A0A0A`
- Nero superficie: `#111111`
- Oro ufficiale: `#D4AF37`
- Oro UI secondario usato in refactor: `#C9A84C`
- Bianco: `#FFFFFF`

## Uso colore

- nero per fondi principali
- oro per accenti, numeri, highlight, badge, stato attivo
- bianco per testo ad alta priorita
- grigi neutri per metadati, ruoli, note secondarie

## Tipografia

Font di riferimento:

- Titoli: stile `Montserrat ExtraBold` / `League Spartan`
- Testi: stile `Inter`

Direzione tipografica corrente:

- una famiglia per i titoli e una per il testo
- titoli compatti, netti, piu solidi che decorativi
- body text pulito, leggibile, con molto respiro
- uppercase solo per kicker, label e metadati

Implementazione v1 corrente:

- display stack: `Montserrat, Inter, system-ui, sans-serif`
- body stack: `Inter, system-ui, sans-serif`
- evitare tracking negativo forte e contrasti tipografici inutili
- preferire gerarchia tramite peso, dimensione e spazio verticale

## UI Language

- spigoli contenuti: radius 8px
- bordi sottili
- interfacce pulite, con piu spazio tra i blocchi
- no card decorative inutili
- no gradienti vistosi se non strettamente funzionali
- una CTA primaria per area, evitando duplicazioni
- home con pochi pulsanti e sezioni compatte
